## 🛡️ 安全注意事项

### ⚡ 电气安全
- 🔌 使用5V低压供电，相对安全
- 💧 避免在潮湿环境操作
- 👀 佩戴防静电手环 (可选)

### 🔧 工具安全
- 🔥 使用烙铁时注意通风
- ✂️ 小心使用剥线钳等工具
- 👓 佩戴护目镜 (推荐)

### 💻 数据安全
- 💾 定期备份代码和数据
- 🌐 使用正版软件
- 🔒 注意隐私数据保护
